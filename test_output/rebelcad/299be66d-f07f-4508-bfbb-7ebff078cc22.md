# EvaluateDerivative

**API Version:** 1.0.0

**Component:** RebelCAD

Computes surface derivatives at given parameter values

## Parameters

- **u**: Parameter in U direction [0,1]
- **v**: Parameter in V direction [0,1]
- **derivU**: Order of U derivative
- **derivV**: Order of V derivative

## Returns

: Derivative vector at (u,v)
