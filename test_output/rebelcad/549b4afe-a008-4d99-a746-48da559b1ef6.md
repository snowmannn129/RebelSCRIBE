# trimLine

**API Version:** 1.0.0

**Component:** RebelCAD

Trims a line at the specified point

## Parameters

- **line**: Line to trim
- **x**: X coordinate of trim point
- **y**: Y coordinate of trim point
- **keepStart**: If true, keeps the portion from start to trim point

## Returns

: New trimmed line

## Exceptions

- **Error**: if trim point is not on the line
