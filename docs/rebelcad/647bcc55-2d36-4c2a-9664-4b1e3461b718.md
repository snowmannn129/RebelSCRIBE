# extendLine

**API Version:** 1.0.0

**Component:** RebelCAD

Extends a line to meet another element

## Parameters

- **line**: Line to extend
- **x**: X coordinate to extend to
- **y**: Y coordinate to extend to
- **fromStart**: If true, extends from start point instead of end point

## Returns

: Extended line

## Exceptions

- **Error**: if extension point is invalid
