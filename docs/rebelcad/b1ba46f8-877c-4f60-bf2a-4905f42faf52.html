ExtendTool

API Version: 1.0.0 | Component: RebelCAD

Provides functionality for extending sketch elements to their intersection points

The ExtendTool class enables extending sketch elements (lines, curves, polygon edges) to meet other elements. It supports: - Line-to-line extension - Line-to-curve extension - Curve-to-curve extension - Polygon edge extension
