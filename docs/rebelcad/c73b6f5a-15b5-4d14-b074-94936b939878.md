# setMirrorLine

**API Version:** 1.0.0

**Component:** RebelCAD

Creates a mirror line from two points

## Parameters

- **x1**: X-coordinate of first point
- **y1**: Y-coordinate of first point
- **x2**: X-coordinate of second point
- **y2**: Y-coordinate of second point

## Exceptions

- **Error**: if points are coincident
