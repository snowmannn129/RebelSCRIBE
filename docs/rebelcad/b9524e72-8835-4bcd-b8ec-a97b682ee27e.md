# ConvertSolidBodyToRawData

**API Version:** 1.0.0

**Component:** RebelCAD

Convert subdivision surface to solid body for boolean operations

/**

## Parameters

- **surface**: Surface to optimize
- **subdivisionLevel**: Level of subdivision to apply before conversion
- **body**: Input solid body
- **detectFeatures**: Whether to detect and preserve sharp features
- **vertices**: Output vertex array (x,y,z coordinates)
- **triangles**: Mesh triangles
- **angleTolerance**: Angle threshold for sharp edge detection
- **indices**: Output triangle index array

## Returns

: Solid body representation
/
    static std::shared_ptr<SolidBody> ConvertToSolidBody(
        std::shared_ptr<SubdivisionSurface> surface,
        size_t subdivisionLevel = 2);
