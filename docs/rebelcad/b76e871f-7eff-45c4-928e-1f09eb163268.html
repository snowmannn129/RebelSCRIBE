ValidateProfile

API Version: 1.0.0 | Component: RebelCAD

Validate the sketch profile is suitable for revolution

Checks: 1. Profile doesn't intersect the rotation axis 2. Profile is on one side of the axis 3. Profile won't self-intersect during revolution

Returns

: true if profile is valid, false otherwise Checks: 1. Profile doesn't intersect the rotation axis 2. Profile is on one side of the axis 3. Profile won't self-intersect during revolution
