MirrorTool

API Version: 1.0.0 | Component: RebelCAD

Tool for creating mirrored copies of sketch entities class: MirrorTool The MirrorTool creates symmetrical copies of sketch entities about a specified mirror line. It supports mirroring of basic entities (lines, arcs, circles) and complex curves (bezier curves, splines). The mirror line can be defined by two points or by an existing line entity.
