BlockAllocator

API Version: 1.0.0 | Component: RebelENGINE

A memory block allocator that efficiently manages variable-sized memory blocks.

The BlockAllocator provides: - Variable block size allocation with cache-friendly alignment - Memory alignment control and optimization - Block coalescing to reduce fragmentation - Thread-safe operations with minimal contention using segmented locks - Memory leak detection and performance monitoring - Fragmentation tracking and management - Proactive memory compaction
